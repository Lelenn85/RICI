Capteur Ultra-Son :

Difficulté : Facile


Matériel :

  • Une platine d'essais.
  • 3 à 5 fils de connexion.
  • Un capteur ultra-son.
  • Une carte Arduino UNO/MEGA.

Voici le montage câblé ci-dessous, il est très simple et propre.

  • Avec 5 fils.
  • Le fil du signal est à câbler sur un PIN PWM.

On peut aussi câbler comme ceci d'une manière un peu moins formelle.

  • Avec 3 fils.
  • Prévoir des fils spéciaux type Mâle/Femelle.
  • Le fil du signal est à câbler sur un PIN PWM.
